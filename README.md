## Paleta de Colores

La paleta ha sido actualizada a tonos púrpura/lavanda combinando con negro:

- **Color Principal**: `#8688BB` (púrpura lavanda)
- **Fondo**: `#F7F6FB` (fondo muy claro)
- **Texto**: `#1A1A1A` (negro puro)
- **Acentos**: Variaciones del púrpura (`#6B6EAA`, `#7A7DB3`)

Todos estos colores están definidos como variables CSS en `css/styles.css` y pueden modificarse fácilmente cambiando los valores en `:root`.

## Cómo Usar

1. **Abrir el sitio**: Abre `index.html` en un navegador
2. **Editar contenido**: Modifica el contenido HTML en `index.html`
3. **Personalizar estilos**: Edita `css/styles.css` para cambios visuales
4. **Agregar interactividad**: Modifica `js/main.js` para nuevas funciones

## Estructura de Archivos

- **HTML**: Toda la estructura y contenido está en `index.html`
- **CSS**: Separado en `css/styles.css` para mejor mantenimiento
- **JavaScript**: Separado en `js/main.js` para mejor organización
- **Imágenes**: Organizadas en `assets/images/`
- **Documentos**: PDFs en `assets/pdf/`
